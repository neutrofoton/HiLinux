1. List file and directory 
   ``` bash
   # Lists files and directories recursively
   $ ls -R

   ```
2. CRUD a file
   ``` bash
   # read a file
   cat file1.txt

   # create a file
   $ cat > file1.txt

   # join multiple file into a file
   $ cat file1.txt  file2.txt > file3.txt
   
   ```
